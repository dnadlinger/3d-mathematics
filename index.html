Mathematische Grundlagen der 3D-Grafik

Fachbereichsarbeit aus Mathematik

David Nadlinger

Februar 2009

Eine aktuelle PDF-Version der Arbeit, die den formalen Anforderungen an eine Fachbereichsarbeit genügt, ist auf GitHub abrufbar. Eine für den Druck optimierte, doppelseitige Variante mit »schönem« Layout befindet sich in Arbeit.

Diese Arbeit entstand unter Verwendung des Open-Source-Satzsystems LaTeX. Bei Interesse können die Quellen ebenfalls auf GitHub eingesehen werden.

Auch die Quellen des Beispielprogramms d4 sind auf GitHub verfügbar.

A paper (in German) about the mathematical basics of 3d computer graphics. Unfortunately, there is no English version available.
